# Kabular

# Welcome to Kabular
Kabular is a miniature one-screen app displaying a table of a typical E-commerce. The data is
obtained via a local Room database.

# Technologies
- Kotlin
- Jetpack compose
- Coroutines
- Jetpack compose
- Room
- MVVM
- Clean Architecture
- SOLID
